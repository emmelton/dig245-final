ABOUT THE PROJECT

This website was built for my DIG-245 course taken Fall semester of 2021. Within the class, we were tasked for our final to create a website that challenged us to further investigate a theme, or a set of themes, from the course. We were asked to create a web application that makes the topic more understandable, relevant, or transparent.

This site employs HTML, CSS, and Javascript code and incorporates skills and themes we have learned within our course this past semester. It is interactive for the user through the button and text area section, where users can interact and input their own text to receive a sentiment analysis.

The user inputs their own data or input and the website transforms the data to reveal insights into the sentiment of the sentence(s) that the API and code received.

WHY THIS DEMO SOFTWARE?

I chose to do a social media comment analysis because I did a court case in high school, through my school and county’s Mock Trial program, on cyberbullying. Although it seems more mild than physical fights from bullying, it is still an issue that is plaguing our society. The court case demonstrated that this issue is far from over since so many people are constantly plugged in and online. Through the concepts and skills I have learned over the semester, a comment review software that helps companies and platforms be aware of the user interactions on their sites might help mitigate the risk and effects of cyberbullying. Although this is just a course project and will not be widely used by any social media platform, I hope it draws attention to how diction in sentences lead to positive and negative sentiment, and to an understanding that negative sentiment can be hurtful and cause pain and problems over time or with continual appearance in someone’s life.
