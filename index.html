How It Works

User Action

A user (you) submits your comment to the company or business offering your honest review and feedback of their product or service.

Code Reaction

Our program code takes the user’s submission and saves it to a database. The submission also activates JavaScript code that runs a sentiment analysis and decides if the submission is positive or negative.

Sentiment Conclusion

If the user’s input was coded as positive, it will return a page designated for postive comments and a few notes on why the comment was positive. If the review was negative, then the program will return a page for negative comments and notes on how this was determined.
